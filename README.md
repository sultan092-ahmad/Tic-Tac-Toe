# 🎮 Tic Tac Toe

A modern and interactive Tic Tac Toe game built using **HTML, CSS, and JavaScript**.

## ✨ Features

- 🎯 Two-player gameplay (Player X vs Player O)
- ✏️ Custom player names
- 🏆 Winner detection with highlighted winning cells
- 🤝 Tie game detection
- 🔄 Restart game functionality
- 🎨 Modern responsive UI
- ⚡ Smooth animations and hover effects

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🎮 How to Play

1. Click **Start Game**
2. Enter player names (optional)
3. Players take turns placing X and O
4. First player to align 3 symbols wins
5. If all cells are filled without a winner, the game ends in a tie
6. Click **Restart** to play again
